This project is a modern, dark-mode, minimalistic landing page designed to be fully responsive across all screen sizes.
It focuses on clean UI, usability, and visual clarity, making it ideal for freelancers and small businesses that want to present their work professionally.

The layout is lightweight and performance-oriented, offering a smooth browsing experience and a strong visual identity.

Key Features

Minimal dark-mode design with light-mode toggle

Fully responsive (mobile, tablet, desktop)

Fast, lightweight, and easy to extend

Designed for freelancers, creatives, and small teams
